Expected File Format

Your file should be in JSONL format (one JSON object per line) or a JSON array:

{"idx": 0, "solved": true, "path_length": 5, "path": [[8, 7], [5, 7], [3, 5], [9, 3], [2, 2]]}
{"idx": 1, "solved": true, "path_length": 3, "path": [[1, 5], [7, 3], [2, 2]]}
{"idx": 2, "solved": false}

Required fields:

  • idx - Presentation index (0-1189 in file, displayed as 1-1190)
  • solved - Boolean (true/false)
  • path - Array of [action_id, resulting_length] pairs (if solved)

Action IDs: 0-11 in file (displayed as h1-h12, see reference below)
